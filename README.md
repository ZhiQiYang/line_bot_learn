# LINE Bot 任務管理系統 - Render 部署版

這是一個專為 Render 平台設計的 LINE Bot 任務管理系統，幫助您管理日常任務、設定計畫、記錄反思。

## 部署到 Render 步驟

### 1. Fork 或克隆此存儲庫

首先，將此存儲庫 fork 到您自己的 GitHub 帳號，或直接下載所有代碼。

### 2. 在 Render 上設置

1. 註冊/登錄 [Render](https://render.com/)
2. 點擊 "New" > "Web Service"
3. 連接您的 GitHub 存儲庫
4. 填寫以下信息：
   - **Name**: 您選擇的服務名稱
   - **Environment**: Python
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `python app.py`

### 3. 配置環境變數

在 Render 的服務設置頁面上，添加以下環境變數：

- `LINE_CHANNEL_ACCESS_TOKEN`: 您的 LINE Channel Access Token
- `LINE_CHANNEL_SECRET`: 您的 LINE Channel Secret
- `USER_ID`: 您的 LINE 使用者 ID
- `PORT`: 設置為 `10000`

### 4. 部署服務

點擊 "Create Web Service" 開始部署。

### 5. 設置 LINE Bot Webhook URL

一旦服務部署完成，您將獲得一個格式為 `https://your-service-name.onrender.com` 的 URL。

在 LINE Developers Console 中設置您的 Webhook URL 為：
`https://your-service-name.onrender.com/callback`

## 使用指南

### 基本指令

- **新增任務**：`新增：看書30分鐘`
- **完成任務**：`完成：看書30分鐘`
- **查詢任務**：`查詢任務`
- **今日進度**：`今日進度`
- **記錄反思**：`反思：今天我學到了...`
- **設定每日計畫**：`設定計畫：{"早上":"晨間閱讀","中午":"午餐後散步","晚上":"復盤一天"}`
- **查看幫助**：`幫助` 或 `help`

### 自動功能

本 Bot 會自動在以下時間點發送訊息：

- **早上 7:00**: 早晨思考問題
- **早上 8:00**: 早上任務提醒
- **中午 12:00**: 中午任務提醒
- **晚上 6:00**: 晚上任務提醒
- **晚上 9:00**: 晚間反思問題

## 維護提示

- Render 免費版會在15分鐘不活動後進入休眠狀態
- 建議使用外部服務（如 UptimeRobot）定期 ping 您的應用程式，保持其活動狀態
- 定期備份 `tasks.json` 和 `reflections.json` 文件，避免數據丟失

## 問題排解

如果遇到問題，請檢查：

1. 環境變數是否正確設置
2. LINE Bot 的 Webhook URL 是否正確
3. 查看 Render 的日誌輸出了解可能的錯誤

## 鳴謝

感謝使用本 LINE Bot 任務管理系統。希望它能幫助您更有效地管理時間和任務！

## 基本功能

使用 `#幫助` 或 `#help` 指令可獲取基本的使用幫助信息，列出所有可用指令。

#幫助

## 時區轉換

時區轉換功能提供了一個方便的方式來計算不同時區間的時間差異。

### 使用格式

```
#時區轉換 [時間] [原時區] to [目標時區]
```

### 示例

```
#時區轉換 14:30 台北 to 紐約
```

### 支持的時區別名

- 台北/台灣: `台北`, `台灣`, `tw`, `taiwan`
- 東京/日本: `東京`, `日本`, `jp`, `japan`, `tokyo`
- 紐約/美國: `紐約`, `美國`, `us`, `usa`, `ny`, `nyc`
- 洛杉磯: `洛杉磯`, `la`, `los angeles`
- 倫敦/英國: `倫敦`, `英國`, `uk`, `london`
- 巴黎/法國: `巴黎`, `法國`, `eu`, `paris`, `france`
- 上海/中國: `上海`, `中國`, `cn`, `china`, `shanghai`
- 新加坡: `新加坡`, `sg`, `singapore`
- 悉尼/澳洲: `悉尼`, `澳洲`, `au`, `australia`, `sydney`
- 柏林/德國: `柏林`, `德國`, `de`, `germany`, `berlin`
- 世界協調時間: `utc`, `gmt`

## 搜尋功能

搜尋功能可讓您快速查詢詞彙定義或主題信息。

### 使用格式

```
#搜尋 [關鍵詞]
```
或
```
#搜索 [關鍵詞]
```

### 示例

```

```

### 搜尋來源

- **中文詞彙**: 使用萌典API查詢中文詞彙定義
- **一般主題**: 使用維基百科查詢主題信息

## 主題地圖

```

```

### 示例

```

記憶術地圖
```

### 可用主題

- 熱力學
- 記憶術

## LIFF 頁面

LINE 前端框架 (LIFF) 頁面提供了更豐富的交互體驗。可通過 Rich Menu 或指令訪問。

## Rich Menu 使用

Rich Menu 是 LINE 機器人的快捷菜單，位於聊天窗口底部。提供了以下功能:

- **新增任務**: 快速創建新任務
- **查詢任務**: 查看當前任務列表
- **今日進度**: 查看當日任務完成情況
- **反思**: 啟動反思問答環節
- **設定計畫**: 設置學習計劃
- **幫助**: 獲取使用幫助

## 系統要求

- LINE 應用程序
- 與機器人建立好友關係

## 隱私與數據

- 您的數據將安全存儲，僅用於提供服務
- 本機器人不會蒐集或分享個人敏感信息

## 常見問題

**Q: 如何添加此機器人為好友?**  
A: 掃描官方 QR 碼或在 LINE 應用中搜索機器人 ID。

**Q: 數據會被保存多久?**  
A: 您的任務和學習數據會被無限期保存，直到您選擇刪除。

**Q: 如何獲取更多幫助?**  
A: 發送 `#幫助` 或 `#help` 獲取指令幫助。
